# ToDo List App (Project Organizer)
A basic project and task organizer built with vanilla Javascript and webpack.
Created as part of The Odin Project full stack developer curriculum
No AI was used in its production.

## Features
- Multiple projects and tasks organization
- Editing information in both projects and tasks
- Persistent storage using Localstorage (autosave on any change)
- Clear out existing storage for a factory reset

## Technology Used
- Javascript (ES6+ Modules)
- Webpack Bundler
- date-fns
- CSS
